# My Mom's VPN

A simple VPN client you can explain to your mom. Supports VLESS/Reality.

---

## ✨ Features

* VLESS support
* Reality support
* Use your own configuration (no account required)
* Simple and clean UI
* Open-source

---

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/your-username/my-moms-vpn.git
```

---

### 2. Add required dependency

This project depends on `libbox.aar`, which is **not included** in the repository.

You need to add it manually:

```
android/app/libs/libbox.aar
```

Make sure your Gradle is configured to load `.aar` files from the `libs` directory.

---

### 3. Run the app

```
npm install
npx react-native run-android
```

---

## ⚙️ Usage

1. Open the app
2. Paste your VLESS configuration
3. Tap "Connect"

That's it.
